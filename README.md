Program ini menentukan apakah bilangan bulat yang dimasukkan pengguna adalah ganjil atau genap. Bilangan genap habis dibagi 2 (sisa 0), ganjil tidak.
Pengguna memasukkan sebuah bilangan bulat.
Gunakan operasi modulo (%) untuk memeriksa sisa bagi dengan 2: jika sisa 0, bilangan genap; jika sisa 1, bilangan ganjil.
Tampilkan hasil penentuan tersebut.
